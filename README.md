# AI-Based Constraint-Aware Inter-City Trip Planner

## Abstract
This project presents an AI-based constraint-aware inter-city trip planner capable of generating feasible and optimized travel routes using graph algorithms and multi-objective optimization techniques. The system considers real-world constraints such as travel cost, time, toll charges, and route feasibility while dynamically re-planning routes when conditions change.

---

## Features
- Constraint-aware route optimization
- Dynamic route re-planning
- Budget and travel-time validation
- Multi-objective optimization
- Intelligent route evaluation
- Feasible path generation
- Graph-based transportation modeling
